# intracloud-iris
Project at IRIS (MediaSchool) for a selfhosted Intranet with multiple services

Folder /tools: All scripts to install each apps
- DokuWiki
- Piwigo
- RoundCube
- Nextcloud
- FreshRSS
- LimeSurvey
- MyBB
- Kanboard
- Kimai - Check README_KIMAI.md for post-install commands
- OsTicket
